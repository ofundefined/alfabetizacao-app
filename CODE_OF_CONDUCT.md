# Code of Conduct

## Our Commitment

We are committed to providing a welcoming, respectful, and inclusive environment for all contributors and users, regardless of background, identity, or experience.

This project is dedicated to adult literacy — a fundamental human right. We operate with the conviction that every person's voice, dignity, and capacity to learn must be honored.

## Our Standards

Examples of behavior that contribute to a positive environment:
- **Respectful dialogue**: Assume good intent, listen actively, ask clarifying questions
- **Centering learners**: Always ask: "Does this serve Sr. Hélio and learners like him?"
- **Humility**: Recognize we are not the experts; learners are
- **Inclusivity**: Welcome contributors from all backgrounds, especially those with lived experience in adult literacy
- **Transparency**: Share decisions, rationale, and feedback openly
- **Accountability**: Take responsibility; admit mistakes; make repairs

Examples of unacceptable behavior:
- Condescension toward learners or contributors
- Discrimination based on background, identity, literacy level
- Harassment or bad-faith engagement
- Spreading misinformation about literacy or pedagogy
- Prioritizing features over learner dignity

## Enforcement

Violations can be reported confidentially to the project lead. We commit to:
- Taking all reports seriously
- Investigating fairly
- Protecting reporters from retaliation
- Imposing consequences proportional to the violation

## Scope

This Code of Conduct applies to:
- All project spaces (GitHub, Discord, etc.)
- Contributor interactions
- User interactions in the app (if social features exist)
- Content (stories, lessons, reflections)

---

**In short**: Treat everyone as you would want to be treated if you were Sr. Hélio learning to read for the first time.
